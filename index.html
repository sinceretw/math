Partial Fraction Decomposition — Problem 9

Decompose: $\displaystyle \frac{x+5}{x^2+7x+12}$
Step 1Factor the denominator
$x^2+7x+12 = (x+3)(x+4)$
Find two numbers that multiply to 12 and add to 7. That's 3 and 4.
Step 2Set up the partial fraction form
$\displaystyle \frac{x+5}{(x+3)(x+4)} = \frac{A}{x+3} + \frac{B}{x+4}$
Each linear factor gets its own term with an unknown constant on top.
Step 3Clear the denominators
$x + 5 = A(x+4) + B(x+3)$
Multiply both sides by $(x+3)(x+4)$ — the fractions disappear.
Step 4Solve for $A$ — let $x = -3$
$-3 + 5 = A(\,-3+4\,) + B(\,-3+3\,)$
$2 = A(1) + B(0) \;\;\Rightarrow\;\; \boxed{A = 2}$
Trick: picking $x=-3$ zeros out the $B$ term, leaving only $A$.
Step 5Solve for $B$ — let $x = -4$
$-4 + 5 = A(\,-4+4\,) + B(\,-4+3\,)$
$1 = A(0) + B(-1) \;\;\Rightarrow\;\; \boxed{B = -1}$
Now $x=-4$ zeros out the $A$ term, so we get $B$ directly.
ResultFinal decomposition
$\displaystyle \frac{x+5}{x^2+7x+12} = \frac{2}{x+3} - \frac{1}{x+4}$
✓ Check: $2(x+4) - (x+3) = 2x+8-x-3 = x+5$  ✓
