Partial Fraction Decomposition — Problem 11

Decompose: $\displaystyle \frac{-5x^5+18x^4-48x^3+134x^2-104x+256}{x^6-4x^5+8x^4-32x^3+16x^2-64x}$
Step 1Check the degrees
Numerator: degree 5.  Denominator: degree 6.
$5 < 6$, so this is a proper rational expression. No long division needed.
Step 2Factor the denominator — pull out $x$
$x^6-4x^5+8x^4-32x^3+16x^2-64x = x\big(x^5-4x^4+8x^3-32x^2+16x-64\big)$
Every term has an $x$, so factor it out first.
Step 3Factor by grouping
$x^5-4x^4+8x^3-32x^2+16x-64$
$= x^4(x-4) + 8x^2(x-4) + 16(x-4)$
$= (x-4)(x^4+8x^2+16)$
Pairs of terms share an $(x-4)$ factor. Group and pull it out.
Step 4Recognize the perfect square
$x^4 + 8x^2 + 16 = (x^2+4)^2$
Think of it as $u^2+8u+16=(u+4)^2$ with $u=x^2$. This is a repeated irreducible quadratic.
Step 5Fully factored denominator
$x^6-4x^5+\cdots-64x = x(x-4)(x^2+4)^2$
Three kinds of factors: a linear $x$, a linear $(x-4)$, and an irreducible quadratic $(x^2+4)$ raised to the 2nd power.
Step 6Set up the partial fraction form
$\displaystyle \frac{N(x)}{x(x-4)(x^2+4)^2} = \frac{A}{x} + \frac{B}{x-4} + \frac{Cx+D}{x^2+4} + \frac{Ex+F}{(x^2+4)^2}$
Rules: Linear factor → constant on top. Irreducible quadratic → linear on top ($Cx+D$). Repeated factor → one term per power.
Step 7Clear denominators
$N(x) = A(x-4)(x^2+4)^2 + Bx(x^2+4)^2$
$\qquad\qquad + (Cx+D)\,x(x-4)(x^2+4) + (Ex+F)\,x(x-4)$
Multiply both sides by $x(x-4)(x^2+4)^2$. Now we have 6 unknowns: $A,B,C,D,E,F$.
Step 8Find $A$ — let $x=0$
$256 = A(-4)(4)^2 + 0 + 0 + 0 = -64A$
$\boxed{A = -4}$
Setting $x=0$ kills every term except the one with $A$. Numerator at $x=0$ is just the constant 256.
Step 9Find $B$ — let $x=4$
$N(4) = -5(1024)+18(256)-48(64)+134(16)-104(4)+256 = -1600$
$-1600 = B(4)(4^2+4)^2 = B(4)(400) = 1600B$
$\boxed{B = -1}$
Plugging $x=4$ makes the $(x-4)$ factor zero, killing the $A$, $C/D$, and $E/F$ terms.
Step 10Find $C, D, E, F$ — match coefficients
Substitute $A=-4, B=-1$ and expand the right side. Compare each power of $x$ to the numerator coefficients on the left.
powerLHSRHS (after subst.)solve
$x^5$$-5$$-4+(-1)+C = -5+C$$C=0$
$x^4$$18$$16+D$$D=2$
$x^3$$-48$$-32+(-8)+0+E$$E=0$
$x^2$$134$$128+8+F = 136+F$$F=-2$
Lower powers ($x^1, x^0$) confirm the values — they're a consistency check, not new equations.
ResultFinal decomposition
$\displaystyle \frac{-5x^5+18x^4-48x^3+134x^2-104x+256}{x(x-4)(x^2+4)^2}$
$\displaystyle = -\frac{4}{x} - \frac{1}{x-4} + \frac{2}{x^2+4} - \frac{2}{(x^2+4)^2}$
✓ Check at $x=1$: LHS $= 251/(-75) = -251/75$.
